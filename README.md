![Free Fire Teleport Banner](https://via.placeholder.com/800x300.png?text=Free+Fire+Teleport+Hack)

- ⏱️ 5-second activation window
- 🔌 No root required
- 📁 Simple configuration file
- 🛡️ Works with most Free Fire versions
